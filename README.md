# EE31 Arduino WebSocket Client

WebSocket-based web application for interacting with Arduino via WS. Final version of code in index.html and index.js

## Quick Start

Need to either conncet to the custom WebSocket server and run it through docker using this command:

docker run -p 8080:8080 gabrielsessions/ee31-server:latest

or you can connect to the class WebSocket server.

## WS URL

Use ws://34.28.153.91 for running with class WS

Use ws://localhost:8080 for running locally with docker container
